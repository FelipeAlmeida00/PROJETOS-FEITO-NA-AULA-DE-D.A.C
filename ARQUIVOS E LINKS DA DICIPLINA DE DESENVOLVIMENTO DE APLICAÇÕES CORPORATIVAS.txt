>> AP1 - Prova Prática <<

_PROVA DE AP1_

AP1 de Desenvolvimento de Aplicações Corporativas

INTRUÇÕES: 

A prova será composta por 5 questões, cada uma delas valendo até 2 pontos;;
Ao finalizar a prova você deve compactar a pasta do projeto no formato .zip ou .rar e enviá-lo como resposta. Cada arquivo compactado, deve ser nomeado da forma: NomeSobrenomeMatricula.zip ou NomeSobrenomeMatricula.rar, por exemplo AntonioMelo123456.rar;
Desse modo, você deve enviar um arquivo para cada uma das duas questões da prova.


QUESTÃO 1:	Crie uma aplicação web no Eclipse estruturando corretamente os pacotes e as classes Model, View e Controller para viabilizar a criação de um CRUDE com banco de dados;

QUESTÃO 2:	Crie, por meio de um script, uma tabela “estoque” no WorkBench do MySQL de modo que as colunas sejam: id_cliente (automática e auto incrementável), nome, idade, endereço e escolaridade. Para isso, escolha os tipos de variáveis adequados para cada campo. Além disso, torne obrigatório a digitação de nome e idade do cliente. Salve o script na mesma pasta do projeto criado no Eclipse.

QUESTÃO 3:	Crie uma página index.html e construa nessa página um botão que, ao ser pressionado, realize a ligação dessa página com o Servlet do projeto.

QUESTÃO 4:	Crie um documento pessoas.jsp contendo um botão "novo contato" que direciona para um arquivo "novo.html" que também deve ser criado. 

QUESTÃO 5:	Faça com que seja possível incluir através da página novo.html um contato no banco de dados MySQL. Para isso crie um Javascript que valide os dados (não deixando ser transmitido dados que tenham nome ou idade vazios)
